В заданной транспортной сети без кратных ребер разрешается задать поток вдоль
только одного пути из вершины 0 в вершину 1. Требуется выбрать путь с наибольшей пропускной способностью и вывести мощность максимального потока, который
можно задать на ребрах данного пути.
Ввод: количество вершин n, матрица n ×n с пропускными способностями ребер.
Вывод: целое число — пропускная способность пути.
